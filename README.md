# AI Internship Projects

This repository contains the projects I completed during my Artificial Intelligence Internship at CODEC Technologies.

During this internship, I learned the basics of machine learning and applied them by building simple, practical projects.

## Projects

### 1. Stock Price Predictor

In this project, I built a basic model to predict stock prices using historical data. It uses simple machine learning techniques like linear regression to understand trends and make predictions.

### 2. Sentiment Analysis on Text Data

This project analyzes text and classifies it as positive, negative, or neutral. It helped me understand how Natural Language Processing (NLP) works in real-world applications.

## Technologies Used

* Python
* Machine Learning
* Basic Data Analysis
* Natural Language Processing (NLP)

## Conclusion

These projects helped me gain hands-on experience in AI and understand how machine learning models are built and used. It was a great learning experience and a good starting point for more advanced topics.
